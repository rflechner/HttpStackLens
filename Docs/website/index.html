See through your HTTP stack.

HttpStackLens is a local HTTP/HTTPS debugging proxy with a live Web UI. Inspect traffic in real time, decrypt HTTPS on demand, and forward through an authenticated corporate proxy — all from a single Go binary.

🖥️ Windows · macOS · Linux ⚡ Single Go binary 🔒 Opt-in HTTPS decryption 🧩 WASM + Tailwind UI
Get started → Explore features
⚠️ For local development only

HttpStackLens is a debugging tool for your own machine. It is not built for production use or shared networks.

What it does

A focused set of tools for understanding what your applications actually send over the wire.

🔀

Forward proxy

Listens on localhost:3128, tunnels HTTPS via CONNECT, and streams requests and responses bidirectionally.

🔓

HTTPS decryption

Opt-in local MITM. Generates per-domain certificates from a debug root CA so you can read encrypted bodies.

📡

Live Web UI

A WASM + Tailwind interface streams traffic over SSE: request list, timings, body decoding and inline image previews.

🏢

Upstream proxy auth

Forward to a corporate proxy with NTLM, Kerberos and Negotiate on Windows — plus a 401-compatibility mode.

💾

Capture & replay

Record sessions to a binary .capture format, browse saved captures and query traffic through a REST API.

🧹

Clean certificate teardown

One click removes the debug CA and every leaf it signed from your OS trust store — nothing left behind.

Quick start

# Build everything (WASM + CSS + native binary)
go run ./build-tools/main.go

# Run the proxy + Web UI
go run .

# Send a request through it
curl -x http://localhost:3128 http://example.com

The proxy listens on localhost:3128 and the Web UI on localhost:9000. See Getting started for prerequisites and build details.
